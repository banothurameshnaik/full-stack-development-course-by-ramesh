Margins:
    Margins are used to create space around elements, outside of any defined borders.
    p {
        margin-top: 100px;
        margin-bottom: 100px;
        margin-right: 150px;
        margin-left: 80px;
    }
    p {
        margin: 100px  80px 100px  150px;
    }

Padding:
    Padding is used to create space around an element's content, inside of any defined borders.
    div {
        padding-top: 50px;
        padding-right: 30px;
        padding-bottom: 50px;
        padding-left: 80px;
    }
